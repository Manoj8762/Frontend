BookCart – Online Bookstore Webpage
I successfully developed the BookCart webpage to address the lack of easily accessible study materials. In today’s digital age, many academic resources are available only through paid subscriptions or rental services. To solve this problem, I created a user-friendly online platform inspired by e-commerce websites.

The project combines my prior knowledge of web development with insights from online prompts and AI tools. I used AI-assisted code generation to implement key features, integrating product listing, search functionality, and a modern interface similar to popular online marketplaces.

This project demonstrates my ability to identify real-world problems and build effective digital solutions using a mix of creativity, technical skills, and AI-enhanced development tools.
